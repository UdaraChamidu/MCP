## create a custom server

- go to settings
- tools and integration
- create new server

```
{
  "mcpServers": {
    "mcp_rag": {
      "command": "uv",
      "args": ["run", "--directory", "C:\\mcp-RAG", "server.py"],
      "env": {}
    }
  }       
}


```

- Then you can see tools in curser settings
- add agent mode in chat
